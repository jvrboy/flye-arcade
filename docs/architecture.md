# FLYE ARCADE Architecture

## Brand direction
- Default dark mode with electric violet, cyan, magenta, and deep-space navy
- Glassmorphism cards with neon edge highlights
- Cinematic landing flow and category portals
- A playful 3D illustration language paired with premium motion graphics

## Cross-platform map
- Web: React + TypeScript + Three.js + React Three Fiber + GSAP + Framer Motion
- Mobile: Expo React Native with branded browsing and category discovery
- Desktop: Tauri shell for the web experience on Windows and macOS
